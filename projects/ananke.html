Project Ananke's Purpose

Project Ananke is currently developing a Thrust Vector Control (TVC) and liquid bi-propellant throttling system for the Rocket Development Lab at Embry-Riddle Aeronautical University. The goal of this project for our system to be utilized in various flight vehicles (FV's) across the organization.
